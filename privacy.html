Privacy Policy

Last updated: June 3, 2026

SINKPESO does not collect, transmit, or share any of your personal data. All data stays on your device. There are no servers, no accounts, and no analytics.

1. Data We Do NOT Collect

SINKPESO is a fully offline application. We do not collect:

  • Personal identification information (name, email, phone)
  • Financial account numbers or credentials
  • Location data
  • Usage analytics or tracking data
  • Cookies or advertising identifiers
  • Any data sent to external servers

2. Data Storage

All data is stored locally on your device using:

  • localStorage — settings, expenses, incomes, bills, budgets, wallets, and vaults
  • IndexedDB — photo diary images and month archives (larger data)

This data never leaves your device. It is not encrypted at rest by the app (your browser's built-in storage security applies). If you set a PIN, it is stored as a SHA-256 hash.

3. Third-Party Services

SINKPESO loads the following resources from CDNs for the user interface:

  • React (unpkg.com) — JavaScript framework for the UI
  • Google Fonts (fonts.googleapis.com) — DM Sans and DM Mono typefaces

These are static file requests. No user data is sent to these services beyond standard HTTP request metadata (IP address, browser user-agent), which is controlled by your browser and network, not by SINKPESO.

4. Data Export & Backup

SINKPESO provides JSON and CSV export features. These exports are generated entirely on your device and saved to your local downloads folder. No data is uploaded anywhere during export.

Encrypted backups (Premium feature) use AES-256-GCM with a password you choose. The encryption happens entirely in your browser using the Web Crypto API.

5. License Keys

If you activate a Premium license, the key is stored in localStorage on your device. License validation happens entirely offline using HMAC-based cryptography. No license key is transmitted to any server.

6. Service Worker

SINKPESO uses a service worker to cache app files for offline use. The service worker only caches static assets (HTML, CSS, JS, fonts). It does not collect or transmit any data.

7. Your Rights

Since all data is stored locally on your device:

  • Access: You can view all your data within the app at any time
  • Export: Use Settings → Export Backup to download all data as JSON
  • Delete: Use Settings → Factory Reset to permanently erase all data, or clear your browser's site data
  • Portability: Export your data as JSON or CSV at any time

8. Children's Privacy

SINKPESO does not collect any data from any user, including children under 13. All data is stored locally on the user's device.

9. Changes to This Policy

If this privacy policy changes, the updated version will be published at this URL with a new "Last updated" date. Material changes will be noted in the app's update notes.

10. Contact

For questions about this privacy policy, open an issue on our GitHub repository.
