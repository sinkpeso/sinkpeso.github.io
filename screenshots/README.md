# Screenshots

This directory contains Play Store screenshots for SINKPESO.

To generate screenshots:

1. Open `sinkpeso.github.io` in Chrome
2. Open DevTools → Toggle Device Toolbar
3. Set dimensions and capture:

**Wide (1280x720):**
- Desktop view of Dashboard
- Save as `dashboard.png`

**Narrow (390x844):**
- iPhone 14 size
- Mobile view of Dashboard
- Save as `mobile-dashboard.png`

Screenshots are referenced in `manifest.json` for PWA install prompts.
